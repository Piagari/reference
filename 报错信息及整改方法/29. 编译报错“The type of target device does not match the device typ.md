29. 编译报错“The type of target device does not match the device type configured by module：xxx”
错误描述
指定target设备的类型与模块配置的设备类型不匹配。
可能原因
指定target设备的类型与模块配置的设备类型不匹配。
解决措施
1.	确保module目录的build-profile.json5文件中targets下指定的设备类型包含所需的设备类型。
2.	确保module目录下src/main/module.json5中配置的deviceTypes字段包含所需的类型。
3.	检查hvigorfile.ts或Hvigor脚本文件是否包含任何可能更改模块deviceTypes设置的代码。
