## 简介

一个基础的仓颉opds（漫画协议）客户端实现

[服务端项目地址](https://gitcode.com/copur/opds_cj/overview)

## 主要参考点

- 发送request请求
- `json.stream`解析配合条件渲染生成页面
- 简单的flex布局
