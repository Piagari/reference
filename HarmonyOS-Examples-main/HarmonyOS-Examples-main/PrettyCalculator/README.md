# Pretty Calculator（我算算）



![Screenshot_2024-09-04T225247](.\assets\icon.png)



## 简介

这是一款简约设计的计算器。具备以下功能：

1、支持四则运算的合法算式的运算，不合法算式会有提示

2、上一个算式的结果可保留至下一个算式

3、合法的算式会记入历史纪录，可调出历史纪录中的算式

4、支持简单的主题切换

5、适配了折叠屏对折和展开的显示
