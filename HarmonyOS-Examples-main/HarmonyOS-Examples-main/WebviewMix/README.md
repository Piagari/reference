# WebviewMix
对 Webview 组件的综合应用，实现了一个小工具&学习平台

## 开发环境

* 操作系统：Windows 11 专业版 23H2
* 开发工具：DevEco Studio NEXT Bet1 5.0.3.900 版本
* 插件：Cangjie Support Plugin(5.0.3.900)
* 编译SDK：5.0.0(12)
* 开发语言：仓颉(.cj)
* node.js：v18.18.2
* hvigor:1.0.5
